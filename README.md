# Kalkulator HPP (Harga Pokok Penjualan)

![Kalkulator HPP App](image.png)

HPP Calculator adalah aplikasi Android modern bergaya *SuperApp* untuk membantu menghitung Harga Pokok Penjualan (HPP) secara mudah dan cepat. Cocok digunakan untuk UMKM, usaha makanan & minuman, toko retail, maupun bisnis rumahan.

Dengan aplikasi ini, pengguna dapat menghitung total modal produksi, biaya bahan baku, kemasan, operasional, hingga menentukan harga jual dan estimasi keuntungan secara otomatis.

## ✨ Fitur Utama
- **Perhitungan HPP Otomatis**: Hitung harga pokok secara akurat berdasarkan komponen biaya spesifik.
- **Kalkulasi Margin & Profit**: Atur persentase margin keuntungan dan dapatkan rekomendasi harga jual terbaik.
- **Simpan Riwayat Perhitungan**: Seluruh riwayat perhitungan tersimpan dengan rapi di perangkat.
- **Mendukung Berbagai Jenis Usaha**: Formulir dinamis yang menyesuaikan input untuk usaha Retail, F&B (Makanan & Minuman), dan Jasa.
- **Ekspor Gambar**: Bagikan hasil simulasi HPP beserta margin keuntungan langsung ke WhatsApp atau media sosial lainnya.
- **Offline Tanpa Internet**: Data tersimpan aman di database lokal (Room SQLite) sehingga aplikasi dapat digunakan kapan saja tanpa internet.

Aplikasi ini dirancang khusus membantu para pelaku usaha menentukan harga jual yang tepat agar bisnis lebih untung, profesional, dan terkontrol.
